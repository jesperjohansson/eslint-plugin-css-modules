Install:

```sh
yarn add --dev @jespers/eslint-plugin-css-modules
```

Configure:

```json
{
  "plugins": ["@jespers/css-modules"],
  "rules": {
    "@jespers/css-modules/no-unused-classes": "error"
  }
}
```
